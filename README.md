Apocalypse Bot (EXTREME)
Configure DISCORD_TOKEN and DISCORD_CHANNEL_ID in your host (Render). Place VHS-like images and audio in the assets/ folder.
Commands (admin):
!alerta  -> enable apocalypse
!silencio -> disable apocalypse
!corromper @member -> corrupt recent messages of member (reposts corrupted)
!interferencia -> send random VHS asset
!statusx -> show status
!sendnow -> send message immediately
